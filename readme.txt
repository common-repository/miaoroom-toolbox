== Description ==

喵容工具箱的核心特色功能， 喵容 WP 聚合 图床 ：喵容工具箱可以帮您自动把 WordPress 中的图片分发上传至图床、自动将文章内的图片替换为图床链接、自动加速媒体库图床、自动修复失效链接，部分图床甚至支持鉴黄、webp等功能。使用图床可以帮小带宽服务器的站长、未备案的站长、国外服务器网站、未使用CDN的站长获得巨大提升。

喵容工具箱为喵容（miaoroom.com）自主开发的一款实用建站插件，在内部自己的网站已经正常运行很长时间。现将其发布给大家一起分享使用。

== 使用注意 ==
<ol>
 	<li>不要在有<strong>重要、隐私图片</strong>的网站上启用本插件，本插件上传至公共图床加速站内图片，并非插件作者的图床。</li>
 	<li>公共图床不保证图片安全性，<strong>本地必须保留备份图片</strong>。</li>
 	<li><strong>严禁上传非法图片至图床</strong>，上传的图片均在公共服务器（非作者服务器）且无法删除。</li>
 	<li>第一正式开启图床后可能会占用服务器带宽传图导致很卡，待图片上传完毕后就没事。上传情况可自行打开地址： <strong>www.域名.com/mrtb/img_host_list</strong></li>
</ol>

<ul>
